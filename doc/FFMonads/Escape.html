Class Method Details

.escape(&block) ⇒ FFMonads::Maybe

Return early from the given block if an evaluated monad has no value.

Examples:

Monads that can be unwrapped will simply run to the end of the block.

monad = some(42)
escape do
  value = monad.v! # monad has a value so it will continue
  some(value + 1)  # it will return some(43)
end

Monads that can’t be unwrapped will return early

monad = none
  escape do
    value = monad.v! # monad does not have a value so it will return none here
    some(value + 1)  # this line will never be reached
  end

Parameters:

  • block (Proc) —

    The block to call.

Returns:

  • (FFMonads::Maybe) —

    The monad returned from the block.

def self.escape(&block)
  block.call
rescue NoValueError => e
  e.source
end
